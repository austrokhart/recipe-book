# Yarn

## Использование

### Обращение к пакетам из CLI

Для единичного обращения к пакетам из CLI необходимо выполнить:

```sh
yarn run <command>
```

Для частого обращения необходимо добавить запись в раздел `scripts` в `package.json`.

```json
{
  "scripts": {
    "<name>": "<command>"
  }
}
```
